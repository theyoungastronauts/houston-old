# Houston CLI


### Setup
```
dart pub global activate dcli
```

### Running
```
    make run
```


### Compile
```
    make compile
```

### CLI Commands

#### Create Module
```
    new_module
```

### Init Feature
```
    new_feature
```
Then edit the `cli/blueprints/{{new_feature}}.yaml` using the `feature.example.yaml` as an example.

### Generate Feature

```
    generate_feature
```
